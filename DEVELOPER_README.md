# Developer README

## Обзор проекта
Children's Literacy Learning Platform - это FastAPI-приложение для обучения детей грамотности с элементами геймификации.

## Архитектура проекта
Проект построен на микросервисной архитектуре с использованием:
- **FastAPI**: Веб-фреймворк для создания API
- **SQLAlchemy**: ORM для работы с базой данных PostgreSQL
- **asyncpg**: Асинхронный драйвер PostgreSQL
- **JWT**: Аутентификация и авторизация
- **WebSocket**: Реал-тайм уведомления

## Структура проекта
```
app/
├── main.py          # Основное приложение FastAPI
├── config.py        # Конфигурация приложения
├── core/
│   └── security.py  # JWT, хэширование паролей
├── db/
│   ├── base.py      # Базовые классы моделей
│   ├── session.py   # Сессия базы данных
│   └── app/models/progress.py  # Модель прогресса уроков
├── models/
│   ├── user.py      # Модели пользователей (Parent, Child)
│   └── curriculum.py # Модели учебного материала (Unit, Lesson, Exercise)
└── services/
    └── gamification_service.py  # Сервис геймификации
```

## Ключевые компоненты

### 1. Аутентификация и авторизация
- Регистрация и вход родителей
- JWT токены (access и refresh)
- Ролевая система (parent/admin)

### 2. Управление пользователями
- CRUD операции для родителей и детей
- Связи между родителями и детьми
- Профили пользователей

### 3. Учебный материал
- Иерархия: Unit -> Lesson -> Exercise
- Публикация/скрытие контента
- Различные типы упражнений

### 4. Геймификация
- XP система
- Уровни и значки
- Стрики активности
- Лидерборд

### 5. Прогресс обучения
- Отслеживание завершения уроков
- Статистика точности
- История активности

### 6. Уведомления
- WebSocket для реал-тайм уведомлений
- Система уведомлений для родителей
- Автоматические уведомления о достижениях

### 7. Администрирование
- Аудит-логи всех действий
- Статистика платформы
- Управление контентом

## API Endpoints
Полный REST API с более чем 30 эндпоинтами:
- `/api/v1/auth/*` - Аутентификация
- `/api/v1/parents/*` - Управление родителями
- `/api/v1/children/*` - Управление детьми
- `/api/v1/units/*` - Учебные блоки
- `/api/v1/lessons/*` - Уроки
- `/api/v1/exercises/*` - Упражнения
- `/api/v1/notifications/*` - Уведомления
- `/api/v1/leaderboard` - Рейтинг
- `/api/v1/admin/*` - Администрирование

## Безопасность

- CORS middleware
- Валидация входных данных через Pydantic
